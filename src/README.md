## 安装步骤
### 基岩版
1. 前提条件：设备上安装了**最新的Minecraft基岩版**。
2. 将下载到的`.mcaddon`文件导入Minecraft中，并在世界中加载
3. 开启世界的假日创作者功能、自定义生物群系、测试版API实验性玩法
4. 大功告成！

### Java版
1. 前提条件：设备上安装了**对应版本的Minecraft Java版及对应的模组加载器**
2. 安装模组到游戏中
3. 大功告成！
